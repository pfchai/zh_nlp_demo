## ChineseNlpCorpus 仓库中的数据集

数据来源 [ChineseNlpCorpus](https://github.com/SophonPlus/ChineseNlpCorpus)

数据获取：

``` bash
cd $ZH_NLP_DEMO_PATH
git clone https://github.com/SophonPlus/ChineseNlpCorpus.git
```

数据说明详见 ChineseNlpCorpus 项目的说明
